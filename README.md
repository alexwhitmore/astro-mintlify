## Running Locally

1. Install dependencies using npm:

```bash
npm install
```

1. Start the development server

```bash
npm run dev
```

## Tech Stack

1. [Astro](https://astro.build/)
2. [Tailwind](https://tailwindcss.com/)
3. [Radix UI](https://www.radix-ui.com/)

## Development

Learn how to preview changes locally

<Callout
  alertDescription='Prerequisite: You should have installed Node.js (version 18.10.0 or higher).'
  variant='info'
/>

Step 1. Install Mintlify on your OS:

<CodeBlock
  tabs={[
    { label: 'npm', code: 'npm i -g mintlify', lang: 'bash' },
    { label: 'yarn', code: 'yarn global add mintlify', lang: 'bash' },
  ]}
/>

Step 2. Go to the docs are located (where you can find mint.json) and run the following command:

<CodeBlock code={`mintlify dev`} lang='bash' />

The documentation website is now available at `http://localhost:3000`.

### Custom Ports

Mintlify uses port 3000 by default. You can use the --port flag to customize the port Mintlify runs on. For example, use this command to run in port 3333:

<CodeBlock code={`mintlify dev --port 3333`} lang='bash' />

You will see an error like this if you try to run Mintlify in a port thatâ€™s already taken:

<CodeBlock
  code={`Error: listen EADDRINUSE: address already in use :::3000
`}
  lang='javascript'
/>

### Mintlify Versions

Each CLI is linked to a specific version of Mintlify. Please update the CLI if your local website looks different than production.

<CodeBlock
  tabs={[
    { label: 'npm', code: 'npm i -g mintlify@latest', lang: 'bash' },
    { label: 'yarn', code: 'yarn global upgrade mintlify', lang: 'bash' },
  ]}
/>
